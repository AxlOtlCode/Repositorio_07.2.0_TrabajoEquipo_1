HTML: ETQUETAS

ETIQUETA DESCRIPCION
!DOCTYPE Indica al naveador que la pagina está basado en htlm
head Contiene toda la información de la pagina que el usuario no ve
body Contiene todo el texto y contenido visible al usuario de la pagina
H1 - H6 Los titulos y subtitulo que se pueden colocar en la pagina
div division en el contenido
img para insertar una imagen
p para parrafos
a para enlaces
br para saltos de linea
hr salto de linea que inserta una linea horizontal
strong para resaltar una palabra o texto
small para hacer un texto mas peueño
table Etiqueta para la aertura de una tabla
thead Indica el bloque de filas que describen las etiquetas de las columnas de la tabla.
tr indica una fia de celdas de la tabla
td define a delda de una columna
th define el encabezado de una tabla
button crea un boton pulsable
